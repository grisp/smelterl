# Task Context: Task 4.12 - Escript exit status forwarding

## Goal
Ensure the shipped `smelterl` escript exits with the same non-zero status returned by command execution failures, so callers can rely on process exit code.

## Scope
- Smelterl escript entrypoint status propagation.
- Regression coverage that executes built escript and asserts non-zero on invalid input.
- Minor version bump for Smelterl.

  `smelterl_cli:run/2`, so command failures propagate through escript process
  exit codes.
- Added regression coverage in `smelterl_plan_generate_SUITE` that executes the
  built escript on invalid motherlode metadata and asserts non-zero exit.
- Bumped Smelterl application version from `0.1.0` to `0.1.1`.
